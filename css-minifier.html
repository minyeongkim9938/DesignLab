📖 사용법 안내

                
            

            

                

                    
🎯 주요 기능

                    
    
                        
  • CSS 압축: CSS 파일을 minified 버전으로 변환하여 파일 크기를 줄입니다.
    • 
                        
  • 파일 업로드: CSS 파일을 드래그 앤 드롭하거나 파일 선택으로 업로드할 수 있습니다.
    • 
                        
  • 코드 입력: CSS 코드를 직접 입력하거나 붙여넣을 수 있습니다.
    • 
                        
  • 압축 통계: 원본 크기, 압축 크기, 압축률을 확인할 수 있습니다.
    • 
                        
  • 다운로드: 압축된 CSS 파일을 다운로드할 수 있습니다.
    • 
                    

                

                

                    
📝 사용 방법

                    
    
                        
  1. CSS 입력:
                            
      
                                
    • 파일 선택 버튼을 클릭하여 CSS 파일을 업로드하거나
      • 
                                
    • 텍스트 영역에 CSS 코드를 직접 입력합니다.
      • 
                            
    
                        
    2. 
                        
  2. 압축하기: "압축하기" 버튼을 클릭하여 CSS를 minified 버전으로 변환합니다.
    3. 
                        
  3. 결과 확인: 압축된 CSS 코드와 압축 통계를 확인합니다.
    4. 
                        
  4. 복사/다운로드: "복사" 버튼으로 클립보드에 복사하거나 "다운로드" 버튼으로 파일을 저장합니다.
    5. 
                    

                

                

                    
💡 팁

                    
    
                        
  • CSS 압축은 공백, 줄바꿈, 주석을 제거하여 파일 크기를 줄입니다.
    • 
                        
  • 압축된 CSS는 읽기 어려울 수 있으므로, 원본 파일은 백업해두는 것을 권장합니다.
    • 
                        
  • 개발 환경에서는 원본 CSS를 사용하고, 프로덕션 환경에서만 minified 버전을 사용하는 것이 좋습니다.
    • 
                        
  • 일부 복잡한 CSS 선택자나 값은 압축 과정에서 문제가 발생할 수 있으니 확인 후 사용하세요.
    •
